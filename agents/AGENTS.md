# AGENTS.md

## 目录定位

- 本目录是一个 GNU Stow 包：`.claude/`、`.codex/`、`.gemini/` 实际会被链接到 `$HOME`，属于**正在生效**的 CLI 配置目录。
- 因此对这些目录下文件的修改，会直接影响本机的 Claude Code / Codex CLI / Gemini CLI 行为；不确定影响时先问清楚再改。

## 常用入口

- Claude Code
  - 规则：`.claude/CLAUDE.md`
  - 配置：`.claude/settings.json`
- Codex CLI
  - 规则：`.codex/AGENTS.md`
  - 配置：`.codex/config.toml`
- Gemini CLI
  - 规则：`.gemini/GEMINI.md`
  - 配置：`.gemini/settings.json`
